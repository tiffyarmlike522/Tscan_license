# Changelog

## 0.2.0-beta

- Added professional dashboard layout with metrics, charts, filters, history, compare, settings, and detail panel.
- Added app grouping: license review, work apps, free/open-source, freemium, developer/runtime, driver/hardware, system/helper, unknown.
- Added license priority labels: Review, Review now, No action, Track terms, Inventory, Identify.
- Added SQLite scan history and session comparison.
- Added Authenticode digital signature checks using hidden batch PowerShell execution.
- Added PDF, XLSX, CSV, and JSON exports.
- Added whitelist/blacklist policy management.
- Added scan progress animation and status phases.
- Reduced noisy false-positive risk findings from helper/system files and weak keywords.

## 0.1.0

- Initial MVP scanner for Windows Registry, Program Files, AppData, and Start Menu shortcuts.
- Added offline JSON license rules, basic risk analyzer, SQLite persistence, and Tkinter UI.
